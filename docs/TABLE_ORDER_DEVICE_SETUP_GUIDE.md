# 테이블 오더 태블릿 설정 가이드

**읽는 시간: 5분**

---

## 🎯 이 문서는 뭔가요?

테이블에 놓을 **주문용 태블릿 25대**를 설정하는 방법입니다.

손님이 태블릿으로 주문하면 → POS에 바로 주문이 들어옵니다.

---

## ✨ 한 줄 요약

```
태블릿에 앱 설치 → POS 주소 입력 → POS에서 테이블 지정 → 끝!
```

---

## 📱 태블릿 설정하기 (25대 전부 이렇게 해요)

### 1단계: 앱 설치하기

1. USB로 APK 파일을 태블릿에 복사
2. 파일 앱에서 APK 파일 찾아서 터치
3. "설치" 버튼 누르기
4. 설치 완료!

---

### 2단계: 앱 실행하고 POS 주소 넣기

1. **Table Order** 앱 터치해서 실행

2. 이런 화면이 나와요:
   ```
   ┌─────────────────────────────┐
   │  🍽️ Table Order Setup      │
   ├─────────────────────────────┤
   │                             │
   │  POS Server Address         │
   │  ┌───────────────────┐      │
   │  │                   │ Test │
   │  └───────────────────┘      │
   │                             │
   │  Table Number               │
   │  ┌───────────────────┐      │
   │  │                   │      │
   │  └───────────────────┘      │
   │                             │
   │  [Reset]  [Save & Start]    │
   └─────────────────────────────┘
   ```

3. **POS Server Address** 칸에 입력:
   ```
   http://192.168.1.100:3088
   ```
   
   > ⚠️ 위 숫자는 **예시**입니다!
   > 
   > 실제 POS 컴퓨터의 IP 주소를 넣어야 해요.
   > 
   > **POS IP 확인 방법:** POS 컴퓨터에서 `ipconfig` 치면 나와요

4. **Test** 버튼 터치
   - ✅ 초록색 체크가 뜨면 성공!
   - ❌ 빨간색 X가 뜨면 → 주소 다시 확인

---

### 3단계: 테이블 번호 넣기 (2가지 방법 중 선택)

#### 방법 A: 직접 입력하기 (간단)

1. **Table Number** 칸에 테이블 번호 입력
   - 예: `T1`, `T2`, `A1`, `B3` 등
2. **Save & Start** 버튼 터치
3. 끝! 바로 주문 화면이 나와요

---

#### 방법 B: POS에서 지정해주기 (추천 ⭐)

**이 방법이 좋은 이유:** 25대를 한 곳에서 관리할 수 있어요!

1. **Table Number는 비워두고** Save & Start 누르지 마세요
2. Test만 하고 그대로 두세요
3. 아래 [POS에서 테이블 지정하기](#pos에서-테이블-지정하기) 섹션으로 가세요

---

## 🖥️ POS에서 테이블 지정하기

### POS 컴퓨터에서 하는 작업입니다

1. POS 프로그램 열기

2. 왼쪽 메뉴에서 차례로 클릭:
   ```
   Hardware Manager → Table Devices
   ```

3. 이런 화면이 나와요:
   ```
   ┌────────────────────────────────────────────┐
   │  📱 Table Devices                          │
   ├────────────────────────────────────────────┤
   │                                            │
   │  ┌──────────────────┐  ┌──────────────────┐│
   │  │ 🟢 TABLET-A1B2   │  │ 🟡 TABLET-C3D4   ││
   │  │                  │  │                  ││
   │  │ 테이블: T1       │  │ 테이블: 미배정   ││
   │  │                  │  │                  ││
   │  │ [배정해제]       │  │ [테이블 배정]    ││
   │  └──────────────────┘  └──────────────────┘│
   │                                            │
   └────────────────────────────────────────────┘
   ```

4. **"테이블: 미배정"** 인 태블릿 찾기

5. **[테이블 배정]** 버튼 클릭

6. 팝업에서 테이블 선택 (예: T1)

7. **[배정하기]** 클릭

8. 끝! 태블릿에 자동으로 적용돼요

---

## 🔄 25대 빠르게 설정하는 방법

1. **25대 전부** 앱 설치
2. **25대 전부** POS 주소만 입력 + Test
3. **POS에서** Table Devices 화면 열기
4. **25대가** 목록에 보임
5. **하나씩** 테이블 배정 (T1, T2, T3...)
6. 끝!

> 💡 태블릿에 스티커 안 붙여도 돼요.
> 
> POS에서 "이 태블릿 = T5" 이렇게 지정하면 되니까요.

## ❓ 자주 묻는 질문

### Q: Test 버튼 눌렀는데 실패해요

**확인할 것:**
1. WiFi 연결됐나요?
2. POS 컴퓨터 켜져 있나요?
3. POS 프로그램 실행 중인가요?
4. 주소에 오타 없나요? (`http://`로 시작해야 해요)

---

### Q: 태블릿이 POS에 안 보여요

**확인할 것:**
1. 태블릿에서 Test 버튼 눌렀나요?
2. 태블릿 WiFi 연결됐나요?
3. POS에서 새로고침 버튼 눌러보세요

---

### Q: 주문했는데 POS에 안 들어와요

**확인할 것:**
1. 태블릿 WiFi 끊겼을 수 있어요
2. POS의 Sales 화면에서 테이블 확인해보세요

---

### Q: 설정 바꾸고 싶어요

**태블릿에서:**
1. 화면 **오른쪽 위 구석**을 **3초 동안** 꾹 누르기
2. 설정 화면이 나와요
3. 수정하고 Save & Start

---

### Q: 태블릿 화면이 자꾸 꺼져요

**해결:**
1. 태블릿 설정 앱 열기
2. 디스플레이 → 화면 자동 꺼짐 → **사용 안 함**

---

### Q: 손님이 다른 앱 실행하면 어떡해요?

**키오스크 모드 설정:**
1. 태블릿 재부팅
2. "홈 앱 선택" 팝업 뜨면 → **Table Order** 선택
3. **항상** 선택
4. 이제 홈 버튼 눌러도 Table Order만 나와요

---

## 📞 도움이 필요하면

1. POS IP 주소 모르겠다 → IT 담당자에게 문의
2. WiFi 연결 안 된다 → 공유기 확인
3. 그래도 안 된다 → 개발팀에 문의
